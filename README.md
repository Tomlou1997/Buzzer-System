# 🎯 抢答系统

基于 **Python + FastAPI + WebSocket** 的局域网抢答软件，管理端和客户端均通过浏览器访问，无需安装任何客户端。

---

## 功能

- **管理端** — 比赛控制、题库管理、选手管理、实时积分榜、设置
- **客户端** — 登录、抢答、答题、实时排名、领奖台
- **题库** — 支持上传 `.xlsx` 格式的 Excel 题库文件
- **实时通信** — WebSocket 实现抢答、计分、倒计时、啦啦队等功能实时同步
- **选手管理** — 在线/离线状态标识、右键菜单设置分数/禁赛/移除
- **积分榜** — 全屏弹窗，实时刷新，支持负数分数
- **啦啦队** — 选手可点击延长答题时间

---

## 技术栈

| 层 | 技术 |
|---|---|
| 后端 | Python 3.13 + FastAPI + Uvicorn |
| 前端 | 纯 HTML / CSS / JavaScript（无框架，单文件） |
| 实时通信 | WebSocket |
| 文件解析 | openpyxl（.xlsx 题库导入） |
| 版本控制 | Git |

## 快速开始

### 1. 安装依赖

```cmd
pip install fastapi uvicorn openpyxl python-multipart
```

### 2. 启动服务

双击 **`启动.bat`**，或命令行执行：

```cmd
cd /d D:\ZM\抢答软件
python server_web.py
```

### 3. 访问

| 端 | 地址 |
|---|---|
| 管理端 | `http://localhost:8888/server` |
| 客户端 | `http://localhost:8888/client` |
| 局域网其他电脑 | `http://<本机IP>:8888/client` |

## 使用流程

1. **导入题库** — 管理端左侧点「题库管理」，上传 `.xlsx` 文件
2. **开始比赛** — 点「🏁 开始比赛」
3. **显示题目** — 点「▶ 下一题」显示第一题
4. **开始抢答** — 点「🚀 开始抢答」
5. **选手答题** — 客户端按空格/回车抢答，选择答案后提交
6. **下一轮** — 自动进入下一题

## 设置项

| 设置 | 说明 | 默认 |
|---|---|---|
| 答对得分 | 回答正确加分数 | 2 |
| 答错扣分 | 回答错误/超时扣分数 | 1 |
| 答题倒计时 | 抢答后答题时间（秒） | 15 |
| 获胜积分 | 达到该分数自动获胜 | 20 |
| 啦啦队次数 | 每名选手每场可使用次数 | 1 |
| 啦啦队时长 | 延长答题时间（秒） | 15 |

## 快捷键

**客户端：**
- `空格` / `Enter` — 抢答（输入框中不触发）
- 点击选项选中，点「提交答案」确认

**管理端：**
- 右键选手 — 设置分数、禁赛、移除选手

## 常见问题

**Q: 其他电脑访问不了？**  
A: 检查防火墙是否开放 8888 端口，确认两台电脑在同一局域网。

**Q: 题库文件格式？**  
A: 支持 `.xlsx` 格式，第一行为表头，需包含题型、题目、选项、答案等列。

**Q: 题库文件格式？**  
A: 支持 `.xlsx` 格式，第一行为表头，需包含题型、题目、选项、答案等列。选项列中的每个选项必须按 `A. 文字` 的格式编写，**字母和文字之间必须是英文句点 `.`**，不能用顿号、空格或其他符号，否则系统无法正确识别答案。例如：

| ✅ 正确格式 | ❌ 错误格式 |
|---|---|
| `A. 北京` | `A、北京` |
| `B. 正确` | `B: 正确` |
| `C. 火药` | `C 火药` |

**Q: 选手断线了怎么办？**  
A: 同名重连即可恢复积分和排名，离线期间积分不变。
