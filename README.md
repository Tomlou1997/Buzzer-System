# 🏆 抢答软件

基于 Python/Tkinter/TCP 的多选手抢答比赛系统。支持自定义题库、自动/手动判题、倒计时、分数排名与获胜排位赛。

## 📦 文件说明

| 文件 | 说明 |
|------|------|
| `server.py` | **主控端（服务端）** — 主持人操作，管理题库、控制抢答、计分判定 |
| `client.py` | **客户端（选手端）** — 选手连接服务器、抢答、选择答案 |
| `launcher.py` | 启动器 — 一键启动主控端或客户端 |

## 🚀 快速开始

### 1. 启动主控端

```bash
python server.py
```

或运行 `launcher.py` 选择「启动主控端」。

### 2. 启动客户端

```bash
python client.py
```

或运行 `launcher.py` 选择「启动客户端」。

> 在同一台电脑测试时，客户端 IP 填 `127.0.0.1` 即可。

## 🎮 使用说明

### 📺 主控端（主持人）

#### ① 导入题库
点击 **📂 导入题库** 或 `Ctrl+O`，支持格式：
- **TXT** — 每行格式 `题目 | 答案` 或 `题目,答案`
- **CSV** — 第一列题目，第二列答案（含表头）
- **JSON** — `[{"question": "...", "answer": "..."}, ...]`
- **XLSX** — 类似 CSV

#### ② 浏览题目
- **上一题 ◀ / 下一题 ▶** — 预览题库
- 边界自动禁用按钮

#### ③ 开始抢答
1. 点击 **开始抢答 🚀** → 所有选手收到题目并激活抢答按钮
2. 选手按下 **空格键** 或 **回车键** 抢答
3. 抢到后选手选择答案 A-F（可多选），点击 **提交答案**

#### ④ 判定得分
- **自动判题**（默认开启）：服务器自动比对答案，即时给分/扣分
- **手动判题**：主持人点击 ✅ **答对给分** 或 ❌ **答错扣分**

#### ⑤ 下一题
点击 **下一题 ▶** 再点 **开始抢答 🚀** 继续比赛。

#### ⑥ 结束本轮
点击 **结束抢答 ■** — 提前结束当前抢答，不判定。

---

### ⚙ 设置面板

点击 **⚙ 设置**，可配置：

| 设置项 | 说明 |
|--------|------|
| **答对加分** | 默认 +2 分（0~999） |
| **答错扣分** | 默认 -1 分（0~999，支持负数效果） |
| **答题倒计时** | 默认 15 秒（5~120） |
| **获胜积分** | 达到此分锁定排名（0=不启用） |
| **自动判题** | 开启后服务器自动比对答案，无需手动判定 |

## 🏆 获胜积分系统（排位赛模式）

### 规则
1. 在 ⚙ 设置中设置目标积分（如 10 分）
2. **第1名**：最先达到 10 分的选手，积分锁定不再变动
3. **第2名**：剩余选手中最先达到 10 分的，积分锁定
4. **第3名**：再下一个达到 10 分的，积分锁定，**比赛结束**
5. 如果选手总数少于 3，产生全部排名后自动结束

### 选手体验
- 获得排名后按钮变灰，提示「已锁定排名」
- 比赛仍在继续时，等待其他选手
- 全部排名产生后，显示最终排名榜

## 📊 积分榜

点击 **📊 积分榜** 按钮 → 全屏显示所有选手当前分数与排名，**🥇🥈🥉** 前三名高亮，已锁定排名显示 `[第1名 ✅]`。

## 🖥️ 客户端操作

### 连接
1. 输入 **选手名**（不能重名）
2. 输入 **服务器 IP**
3. 点击 **连接服务器**

### 抢答
- 按 **空格键** 或 **回车键** 抢答（全局快捷键）
- 抢到后选择 **A~F**（可多选），点击 **提交答案**

### 快捷键
| 按键 | 功能 |
|------|------|
| `空格` | 抢答 |
| `回车` | 抢答 |
| `F11` | 全屏 |
| `Esc` | 退出全屏 |

### 断线提示
关闭窗口时会提示：「退出视为放弃比赛资格，确认要退出吗？」

## 🔄 其他功能

### 🔄 重置计分
点击 **🔄 重置计分** → 弹窗确认 → 所有选手分数归零，清除已锁定排名，比赛重新开始。

### 日志
- 主控端右侧信息区显示操作日志（限 8 行）
- 调试日志写入 `server_debug.log`（毫秒级）

## 📝 题库格式示例

### TXT 格式
```
中国共产党成立于哪一年？ | 1921
抗日战争胜利是哪一年？ | 1945
以下哪些是中国的直辖市？ | ABC
```

### CSV 格式
```csv
题目,答案
中国共产党成立于哪一年？,1921
抗日战争胜利是哪一年？,1945
```

### JSON 格式
```json
[
  {"question": "中国共产党成立于哪一年？", "answer": "1921"},
  {"question": "抗日战争胜利是哪一年？", "answer": "1945"}
]
```

> **答案格式**：单选填答案本身（如 `1921`），多选填选项字母（如 `ABC`），不区分大小写。

## 🌐 网络说明

- 默认端口：**8888**
- 本机测试 IP：**127.0.0.1**
- 局域网测试：主控端 IP 用 `ipconfig` 查看，客户端填对应的 IP
- 心跳保活：每 5 秒发送心跳包检测连接
